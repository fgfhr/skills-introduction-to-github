Git 核心5大命令速查表

&#x20;1. git clone

作用：将GitHub远程仓库下载到本地

示例：git clone https://github.com/fgfhr/skills-introduction-to-github.git



2\. git add .

作用：把当前目录所有修改文件加入暂存区



&#x20;3. git commit -m "备注文字"

作用：本地打包保存修改，-m 填写本次更新说明



4\. git pull origin main

作用：拉取GitHub网页最新文件，防止推送冲突



&#x20;5. git push origin main

作用：把本地文件上传同步到GitHub远程仓库
